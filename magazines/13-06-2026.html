Проверено 13 источников за период. Найдено 3 релевантных обновления. Затронуты платформы: Яндекс (Adfox), VK (ОРД), MAX.

Главное
Adfox: обновлены монетизация, отчетность и интерфейс
Яндекс / Adfox 11 июня 2026 Яндекс Реклама (официально)
Что изменилось
Яндекс опубликовал дайджест обновлений Adfox. Изменения сразу в нескольких частях платформы:
  • Новый интерфейс монетизации: РСЯ и внешние сети управляются в одном разделе
  • Метрика "Прогноз выполнения" для кампаний с цветовой индикацией рисков
  • Аудиторные отчеты по частоте и охвату, сегменты Яндекс Крипты для таргетинга
  • Новый API v2 статистики, работает параллельно со старым до августа 2026
  • Конструктор отчетов теперь доступен на мобильных устройствах
  • Обновленный раздел недокрутов с иерархией суперкампания / флайт
  • Автоматизация отчетности в ЕРИР с массовой отправкой актов
Почему важно
Одно из самых крупных обновлений Adfox за последнее время. Затрагивает монетизацию, аналитику, интерфейс недокрутов и операционную работу с ЕРИР. Срок поддержки старого API ограничен: переход на v2 нужно завершить до августа 2026.
Кому полезно
Паблишерам на Adfox, программатик-специалистам и командам, у которых есть интеграция через Adfox API.
Если есть интеграция через API, миграция на v2 приоритетна до августа. Остальные изменения, прежде всего новый интерфейс монетизации и аудиторные отчеты, стоит проверить в кабинете в ближайшее время.
Остальные обновления
ОРД VK: появилось автопродление тарифов
VK / ОРД 10 июня 2026 Click.ru
Что изменилось
В ОРД VK добавлена возможность автоматического продления тарифа. По умолчанию функция отключена. Для активации нужно перейти в раздел управления оплатой, выбрать тариф и включить опцию. Отключить можно в любой момент до следующего списания.
Кому полезно
Агентствам и рекламодателям, которые используют ОРД VK для маркировки рекламы. Особенно актуально для команд с большим объемом кампаний, где ручное продление создает операционный риск.
Небольшое, но полезное изменение. Прерывание доступа к ОРД влечет технические проблемы с маркировкой. Стоит включить автопродление и убедиться, что на балансе достаточно средств.
Рынок рекламы в MAX вырос в 55 раз за полгода
MAX 8 июня 2026 Cossa / Telega.in + АРИР
Что показало исследование
Первое публичное исследование рекламного рынка MAX. С октября 2025 по март 2026 объем вырос в 55 раз. Крупнейшие категории: образование (33,4%), блогеры (25,5%), недвижимость (11,8%). Половина рекламодателей тратит менее 10 млн рублей в год. Прогноз на конец 2026: 2-2,5 млрд рублей.
Барьеры для входа
Главные трудности по данным опроса: нехватка кейсов (48%), сложность подбора каналов (43%).
Цифра 55 раз отражает эффект низкой базы, но тренд реальный. Средние бюджеты невелики, рынок ещё не насыщен. Для рекламодателей в образовании или недвижимости MAX сейчас - канал с относительно низкой конкуренцией.
Не вошло в выпуск
  • VK Tech запустил VK AI Researcher для автоматизации маркетинговых исследований - вне окна дат, первая публикация 25 мая; инструмент для исследований, не рекламный кабинет
  • VK: финансовые результаты за I квартал 2026 года - вне окна дат, финансовая отчетность компании без продуктовых изменений
  • Вебинар: реклама в MAX, как бизнесу стартовать - анонс мероприятия, не продуктовое обновление
  • Стоит ли производителям уходить с маркетплейсов в 2026 году - аналитика рынка без изменений рекламного инструмента
Summary
  • Adfox выпустил комплексное обновление: новый интерфейс монетизации, метрика прогноза выполнения, аудиторные отчеты, API v2. Командам с интеграцией через API нужно завершить переход до августа 2026.
  • ОРД VK добавил автопродление тарифов. Операционно полезная мелочь для агентств: снижает риск прерывания маркировки из-за забытого продления.
  • Первое публичное исследование рынка MAX дает ориентиры: 55-кратный рост с низкой базы, невысокие средние бюджеты, конкуренция ещё не высокая. Полезно для планирования экспериментов с каналом, особенно в образовании и недвижимости.
