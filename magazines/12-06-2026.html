Главное
Google Chrome 150 отключит блокировщики рекламы в конце июня
Google Chrome AdIndex 11 июня 2026
Что произошло
Google переводит Chrome на Manifest V3. С выходом Chrome 150 в конце июня 2026 года браузер прекратит поддержку расширений на Manifest V2 - к ним относятся uBlock Origin и большинство популярных блокировщиков рекламы. Временное исключение сделано только для Enterprise. Остальным пользователям предложены Firefox и Chromium-форки как альтернативы.
Почему важно
У миллионов пользователей Chrome исчезнут инструменты блокировки рекламы. Это увеличит видимую аудиторию для медийки, видео и контекста. Вероятна корректировка CPM по мере изменения объёма доступного инвентаря.
Кому следить
Всем, кто ведёт кампании с оплатой за охват в Chrome-трафике. Особенно актуально для CPM-закупок, видеорекламы и паблишеров с монетизацией через баннеры.
Вывод С конца июня охват в Chrome вырастет органически. Следите за динамикой CPM и CTR в первые недели после выхода Chrome 150 - возможны заметные изменения в статистике кампаний.
Яндекс Директ: гостиницы и туристические сервисы теперь в тематическом блоке отелей поиска
Яндекс Директ Яндекс Реклама 11 июня 2026
Что изменилось
Гостиницы, хостелы, санатории, апартаменты, глэмпинги, базы отдыха и агрегаторы теперь могут занимать первые позиции в тематическом блоке отелей в поиске Яндекс. Настройка ведётся через Единые перфоманс-кампании в Директ Pro (профессиональный режим). Данные об организации автоматически берутся из профиля в Яндекс Бизнесе.
Охват формата
Тематический блок появляется в ответ на более чем 80% запросов по теме отелей. Каждый второй такой запрос заканчивается кликом по блоку. На десктопе - до 3 объявлений, на мобильном - до 7, на карте - до 10.
Акция до 31 августа
Промокод YNDXHOTEL даёт +50% к пополнению счёта (минимальное пополнение 5 000 руб., максимальный бонус 10 000 руб.).
Вывод Если в портфеле есть клиенты из туристической вертикали - проверьте готовность их профиля в Яндекс Бизнесе и запустите тест до конца августа, пока действует бонус к пополнению.
Остальные обновления
Adfox: большое обновление монетизации, отчётности и таргетинга
Adfox Яндекс Реклама 11 июня 2026
Что изменилось
Сразу несколько обновлений: переработан интерфейс настроек монетизации с группировкой по форматам РСЯ и Header Bidding; прогноз выполнения кампаний заменил показатель "Соответствие плану" - теперь с цветовой индикацией (зелёный / оранжевый / красный); добавлены отчёты по частотному распределению пользователей и охвату кампании; в конструктор отчётов добавлен VTR; конструктор отчётов теперь работает на мобильных устройствах.
Дедлайн по API
Запущен Statistics API v2. Поддержка API v1 прекращается 1 августа 2026 - все интеграции необходимо перевести на v2 заранее.
Нацеливание без cookie
Новые методы таргетинга: через параметры URL и идентификаторы устройств - для работы в средах без cookie и в WebView.
Вывод Если используете Adfox через API - миграция на v2 обязательна до 1 августа. Для работы с монетизацией рекомендуется ознакомиться с обновлённым интерфейсом.
