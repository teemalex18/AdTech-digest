В медийных кампаниях Директа появилась возможность увеличивать частоту показов на пользователя
14.05.2026
Яндекс Директ Источник: ppc.world ✔Подтверждено: Yandex Adv
Что изменилось
В медийных кампаниях Директа добавлена функция управления частотой показов на одного пользователя. Опция работает только в связке со стратегией «Максимум показов по минимальной цене».
Почему важно
Рекламодатели получили инструмент для повышения охватного давления на целевую аудиторию — можно добиться большего числа контактов с одним пользователем без смены стратегии.
Кому полезно
Бренд-менеджеры и медиапланеры, запускающие охватные медийные кампании; агентства, ведущие имиджевое продвижение крупных клиентов.
Вывод
Новая настройка расширяет контроль над медийными размещениями в Директе и позволяет точнее управлять частотой контакта с аудиторией. Доступна только при использовании стратегии «Максимум показов по минимальной цене».
Яндекс Директ запустил автоподбор Telegram-каналов по описанию контента
13.05.2026
Яндекс Директ Источник: Yandex Adv ✔Подтверждено: ppc.world, SEOnews
Что изменилось
Добавлен новый способ автоподбора Telegram-каналов для размещения рекламных постов через Директ — на основе текстового описания желаемого контента. Система сама находит каналы, подходящие под заданные параметры, в том числе на стыке тематик.
Почему важно
Позволяет находить релевантные каналы без ручного перебора, особенно эффективно для нишевых или смежных аудиторий, которые сложно охватить стандартными категориями.
Кому полезно
Специалисты по performance и influence-маркетингу, агентства, рекламодатели с узкой целевой аудиторией.
Вывод
Новый инструмент снижает трудозатраты на подбор каналов и повышает точность таргетинга при размещении в Telegram через Директ. Особенно полезен для тематик, охватывающих несколько сегментов аудитории.
Остальные обновления
Яндекс Метрика позволила строить воронки по ecommerce-событиям, звонкам, статусам заказов из CRM и офлайн-конверсиям
13.05.2026
Яндекс Метрика Источник: Yandex Adv ✔Подтверждено: ppc.world, SEOnews
Что изменилось
В Яндекс Метрике расширены условия построения воронок: теперь доступны ecommerce-события, заказы из CRM, звонки, чаты и любые другие офлайн-конверсии. Ранее воронки строились преимущественно по онлайн-событиям на сайте.
Почему важно
Рекламодатели могут отслеживать полный путь пользователя — от клика по рекламе до фактической оплаты и закрытия заказа, включая офлайн-этапы. Это даёт более точную оценку эффективности рекламных кампаний.
Кому полезно
Аналитики, e-commerce специалисты, рекламодатели с длинным циклом сделки, бизнесы с омниканальными продажами.
Вывод
Обновление делает воронки в Метрике по-настоящему сквозными и применимыми для бизнесов с офлайн-составляющей. Теперь можно оценивать рекламу не только по онлайн-метрикам, но и по реальным продажам.
Wildberries запустил аналитического ИИ-ассистента для всех продавцов
12.05.2026
Wildberries Источник: ppc.world ✔Подтверждено: SEOnews
Что изменилось
Wildberries запустил ИИ-ассистента для продавцов в личном кабинете. Инструмент показывает динамику продаж, отзывы и позиции товаров в выдаче маркетплейса.
Почему важно
Селлеры получают автоматизированную аналитику прямо внутри кабинета без необходимости использовать сторонние сервисы — это ускоряет принятие решений по управлению ассортиментом и рекламой.
Кому полезно
Продавцы Wildberries, менеджеры маркетплейсов, e-commerce специалисты.
Вывод
Новый ИИ-ассистент упрощает работу с аналитикой для продавцов WB и снижает зависимость от внешних инструментов. Полезно прежде всего для малого и среднего бизнеса без выделенных аналитиков.
Владельцы российских Telegram-каналов могут выводить вознаграждение за рекламу в TON
14.05.2026
Telegram Ads Источник: ppc.world
Что изменилось
Авторы российских Telegram-каналов получили возможность выводить вознаграждение за показ рекламы в криптовалюте TON. Опция действует только в том случае, если реклама показывалась иностранной аудитории.
Почему важно
Расширяются варианты монетизации для авторов каналов с международной аудиторией — появляется альтернативный способ вывода средств в дополнение к существующим.
Кому полезно
Владельцы Telegram-каналов с иностранной аудиторией, участвующие в программе монетизации Telegram.
Вывод
Новая опция актуальна для авторов, чья аудитория выходит за пределы России. Для рекламодателей это не меняет процесс закупки рекламы, но косвенно повышает привлекательность площадки для крупных авторов.
В Adstat.pro появился инструмент автоматизации «Стратегии 2.0»
13.05.2026
Adstat.pro Источник: Magnetto
Что изменилось
На платформе Adstat.pro запущен инструмент «Стратегии 2.0», предназначенный для автоматизации управления рекламными размещениями. Детали функциональности доступны внутри материала.
Почему важно
Автоматизация стратегий управления рекламой на уровне платформы снижает ручную работу специалистов и потенциально повышает эффективность размещений.
Кому полезно
Специалисты и агентства, работающие с Adstat.pro для управления рекламой в Telegram и других каналах.
Вывод
Обновление расширяет возможности автоматизации в Adstat.pro. Рекомендуется ознакомиться с деталями функции напрямую на платформе для оценки применимости в конкретных задачах.
Что это значит для работы
  • Медийная реклама в Директе получила новый рычаг управления охватным давлением. Если ведёте имиджевые кампании по стратегии «Максимум показов по минимальной цене» — стоит протестировать новую настройку частоты.
  • Автоподбор Telegram-каналов по описанию снимает рутину ручного поиска и открывает доступ к нишевым аудиториям на стыке тематик. Актуально для клиентов с узким позиционированием.
  • Расширение воронок в Метрике — важный шаг для бизнесов с офлайн-продажами и длинным циклом сделки. Теперь сквозная аналитика стала доступнее без сторонних инструментов.
  • ИИ-ассистент Wildberries полезен продавцам, у которых нет выделенного аналитика — базовую сводку по продажам и позициям теперь не нужно собирать вручную.
  • Вывод вознаграждений Telegram Ads в TON касается прежде всего авторов с зарубежной аудиторией. Рекламодателям изменение процессов не затрагивает, но стоит держать в уме при работе с крупными каналами.
Не вошло в выпуск
За отчётный период отброшенных материалов с чёткими данными не зафиксировано — конкретные источники для исключения определить не удалось. 3 источника из 11 не были охвачены мониторингом в данном окне.
