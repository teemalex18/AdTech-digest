Главное
Дайджест Adfox: обновлённый интерфейс монетизации, нацеливание без cookie, VTR и новые отчёты
Яндекс Реклама Adfox 11 июня 2026
Что изменилось
Яндекс выпустил масштабный дайджест обновлений Adfox - 10 новых инструментов и изменений:
  • Новый интерфейс монетизации: поиск по названию в Header Bidding, фильтрация по статусу и типу подключения, настройки на уровне отдельных площадок.
  • Прогноз выполнения кампании с цветовой индикацией: зелёный (95%+), оранжевый (80-94%), красный (меньше 80%) - с учётом темпа, истории и конкурентных кампаний.
  • Новые аудиторные отчёты: распределение по частоте просмотров и охват кампании.
  • Сегменты Яндекс Крипты в конструкторе отчётов - оценка аудиторного состава инвентаря по интересам.
  • Нацеливание без cookie: тестирование показа в браузерах с cookie, через URL-параметры и на конкретных устройствах.
  • Метрика VTR для видеокампаний в конструкторе отчётов на уровне флайта.
  • Новый API v2 для статистики. Старый API будет отключён 1 августа 2026 года.
  • Мобильная версия конструктора отчётов - весь функционал десктопа на телефоне.
  • Обновлённая вкладка «Недокруты»: двухуровневая иерархия Суперкампания/Флайт, новые колонки, массовая выгрузка в Excel.
  • Автоматизация актов для ЕРИР: проверка и массовая отправка без скачивания, удаление актов из интерфейса.
Кому важно
Паблишерам и агентствам, работающим с монетизацией через Adfox. Отдельно - командам с интеграцией статистики через API.
Миграция на API v2 обязательна до 1 августа 2026 - старый API будет отключён. Проверьте наличие интеграций со старым API в ваших системах и включите обновление в план на июль.
Яндекс Директ: отели и туристические сервисы занимают первые позиции в блоке отелей поиска
Яндекс Реклама Яндекс Директ 11 июня 2026
Новая возможность
Отели и туристические сервисы теперь могут занимать верхние позиции в блоке отелей поисковой выдачи через рекламу в Яндекс Директ. Формат позволяет управлять стоимостью бронирований и привлекать клиентов напрямую из поиска.
Кому подходит
Отелям, санаториям, базам отдыха, туристическим агентствам и онлайн-сервисам бронирования.
Новый рекламный формат для туристического вертикала в Директе - стоит протестировать для отельных клиентов. Блок отелей в поиске конкурирует напрямую с органическими результатами и агрегаторами.

Google отключает все блокировщики рекламы в Chrome
AdIndex Chrome 11 июня 2026
Что произошло
Google отключил поддержку расширений-блокировщиков рекламы в браузере Chrome. Пользователи, ранее скрывавшие рекламу с помощью расширений, теперь видят медийную рекламу в полном объёме.
Что это значит для рекламы
Потенциальное расширение охвата display-кампаний: часть аудитории, которая ранее не видела баннеры, возвращается в зону показов. Конкретный эффект зависит от доли пользователей с блокировщиками в целевой аудитории.
Стоит отследить динамику показов и CTR в медийных кампаниях в ближайшие недели - эффект может быть заметен в сегментах с активными пользователями Chrome.
Роль поискового ранжирования при отборе сайтов в ответы Алисы существенно выросла
Cossa.ru Яндекс / Алиса 11 июня 2026
Суть исследования
По данным «Ашманов и партнёры», поисковое ранжирование в Яндексе стало значительно более важным фактором при отборе источников для ответов Алисы. Сайты с высокими SEO-позициями с большей вероятностью попадают в AI-ответы ассистента.
Для рекламной стратегии
Бренды с активной SEO-стратегией в Яндексе получают дополнительное преимущество: органические позиции влияют на видимость в AI-поиске. Это аргумент в пользу интегрированной стратегии - платный трафик плюс органика.
Прямого изменения в рекламных кабинетах нет, но данные полезны для обоснования SEO-инвестиций клиентам, которые работают в Яндексе с фокусом только на платном трафике.
